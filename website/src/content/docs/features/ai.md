## Overview

OpenPMS includes 8 AI-powered features that help automate and optimize your rental operation. All AI features use a **Bring Your Own Key (BYOK)** model — you provide your OpenAI or Anthropic API key and pay only for what you use.

## The 8 AI Features

### 1. Smart Pricing

Analyzes historical occupancy, local events, seasonality, and competitor rates to suggest optimal nightly prices. Review and apply suggestions per property or in bulk.

### 2. Guest Message Generation

Drafts personalized guest messages based on context (pre-arrival, during stay, post-checkout). Adapts tone and language to the guest's profile.

### 3. Review Response Writer

Generates professional responses to guest reviews. Matches the tone to the review sentiment — grateful for positive reviews, empathetic and solution-oriented for negative ones.

### 4. Listing Optimization

Analyzes your property listing and suggests improvements to title, description, amenities highlighting, and photo ordering to improve conversion rates.

### 5. Document OCR

Extracts structured data from ID cards and passports during the guest check-in process. Supports Portuguese CC, EU ID cards, and international passports.

### 6. BI Chat

Natural language interface to your analytics data. Ask questions and get answers with charts. See the [Analytics](/features/analytics/) page for details.

### 7. Maintenance Predictor

Analyzes task history and guest feedback to predict upcoming maintenance needs. Flags properties likely to need attention before issues escalate.

### 8. Expense Categorizer

Automatically categorizes expenses from receipts and bank statements into accounting categories. Reduces manual bookkeeping work.

## Budget Tracking

Set a monthly AI spending limit in Settings. The system tracks token usage across all features and pauses AI calls when the budget is reached. Usage breakdown is visible per feature.

## Configuration

Go to **Settings > AI** to add your API key, select the default model, and set your monthly budget. Each AI feature can be individually enabled or disabled.
